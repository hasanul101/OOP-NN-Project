# OOP-NN-Project

This project implements a basic Tensor class with autograd functionality.

### Current features:
- Scalar tensor operations
- Gradient tracking
- Backward propagation
- Linear function demonstration

### Next steps:
- Extend to vector operations
- Build simple neural network components

- Final result:
w ≈ 2.0
b ≈ 1.0
Loss decreases over epochs → model converges

I implemented a minimal tensor engine with automatic differentiation and used it to train a linear regression model using gradient descent. The model successfully learns parameters by minimizing loss over multiple epochs.
